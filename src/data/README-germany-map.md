# Germany Bundesländer map data

- **File:** `germany-bundeslaender-map.json`
- **Source GeoJSON:** [deutschlandGeoJSON](https://github.com/isellsoap/deutschlandGeoJSON) — `2_bundeslaender/4_niedrig.geo.json` (simplified boundaries).
- **Processing:** Each Land’s largest polygon ring was projected (equirectangular) into SVG `viewBox="0 0 520 560"`; centroid of that ring is used for animation hubs.
- **Attribution:** Respect the license of the upstream repository and, where required, **© GeoBasis-DE / BKG** for German administrative boundaries.

This is **not** the commercial [Vemaps](https://vemaps.com/germany) asset; it is independently derived from open geographic data for a similar visual result.
